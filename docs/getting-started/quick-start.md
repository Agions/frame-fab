## 步骤 1「提交你的故事」

### 输入方式

**方式一：直接输入**
- 在文本框中输入故事大纲或完整剧本
- 支持中文、英文、日文等多语言内容

**方式二：文件上传**
- 支持格式：`.txt`、`.docx`、`.md`、`.pdf`
- 点击上传按钮选择文件

### 输入示例

```
在一个未来城市，机器人与人类共同生活。
主角小明是一台拥有自我意识的家务机器人，
他渴望了解什么是"家"的意义...
```

### 设置选项

| 选项 | 说明 |
|------|------|
| 标题 | 为你的漫剧设置一个名称 |
| 时长 | 预估时长（影响生成画面数量） |
| 风格 | 科幻 / 唯美 / 热血 / 搞笑 |

## 步骤 2「AI 无人值守」

### 全自动处理流程

```
故事输入
    │
    ▼
┌─────────┐
│ 剧本生成 │ ◀── AI 分析故事结构
└─────────┘
    │
    ▼
┌─────────┐
│ 分镜设计 │ ◀── AI 拆分场景与镜头
└─────────┘
    │
    ▼
┌─────────┐
│ 画面生成 │ ◀── AI 文生图技术
└─────────┘
    │
    ▼
┌─────────┐
│ 旁白合成 │ ◀── AI 文字转语音
└─────────┘
    │
    ▼
┌─────────┐
│ 视频封装 │ ◀── 自动剪辑合成
└─────────┘
    │
    ▼
```

### 处理状态

| 状态 | 说明 |
|------|------|
| ⏳ 等待中 | 任务排队中 |
| 🔄 处理中 | 各环节进行中，可实时预览 |
| ✅ 已完成 | 可下载查看 |
| ❌ 失败 | 可查看错误日志重试 |

## 步骤 3「下载成片」

### 支持格式

| 格式 | 适用场景 | 文件大小 |
|------|----------|----------|
| MP4 | 主流平台分享 | 中等 |
| GIF | 社交媒体/聊天 | 较小 |
| 序列帧 | 进一步编辑 | 较大 |

### 预览与下载

- 生成过程中可实时预览画面
- 支持在线播放确认效果
- 一键复制分享链接
- 下载到本地保存
