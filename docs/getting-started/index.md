## 🎬 快速开始

### 第一步：上传你的故事

支持三种输入格式：
- 📚 **小说文本** — 直接粘贴或上传小说全文
- 🎬 **剧本格式** — 结构化的剧本文件
- 💬 **需求描述** — 用自然语言描述你的想法

### 第二步：选择模式与风格

| 模式 | 说明 | 适合场景 |
|------|------|----------|
| **Autonomous Mode** ✅ | 全自动，一键开始，无需任何操作 | 快速成片、批量生产 |
| **Manual Mode** | 逐环节审批，可精细调整 | 定制化创作、特定需求 |

选择画风：2D / 3D / 动漫 / 写实

### 第三步：点击「开始」

等待 AI 完成全部制作，收到通知后预览并下载成片！

## ⚖️ Autonomous Mode vs Manual Mode

### Autonomous Mode（全自动模式）

```
用户操作：提供原材料 → 按「开始」→ 等待 → 下载成片
AI 行为：自主分析 → 自主生成 → 自主审核 → 自主修复（如需）
```

**核心特性：**
- 🕐 **省时高效** — 全程无需人工干预
- 🧠 **AI 自审** — 每步自动检查质量，不合格自动修复
- 💾 **断点续传** — 支持中途暂停、刷新继续
- 🔄 **自动降级** — 主模型不可用时自动切换备选

### Manual Mode（手动模式）

```
用户操作：导入 → 分析（确认）→ 脚本（编辑）→ 分镜（调整）→ ...
```

**核心特性：**
- ✏️ **精细控制** — 每步均可手动调整
- 🎨 **高度定制** — 满足特定创作需求
- 👀 **全程可见** — 每步输出均可审核

## 📋 步骤概览

### 全自动流水线（11 步）

| 步骤 | 名称 | 说明 | 自主模式 |
|------|------|------|----------|
| 1 | ImportStep | 解析原材料，切分章节 | Auto |
| 2 | AnalysisStep | 分析故事结构、人物、场景 | Auto |
| 3 | ScriptStep | 生成结构化视频剧本 | Auto |
| 4 | CharacterStep | 创建角色设定卡 | Auto |
| 5 | SceneStep | 场景规划 | Auto |
| 6 | StoryboardStep | 生成分镜脚本 + 参考图 | Auto |
| 7 | RenderStep | 批量渲染关键帧 | Auto |
| 8 | VideoEditStep | 视频剪辑 + 转场 | Auto |
| 9 | AudioStep | 配音 + 音效 + 唇形同步 | Auto |
| 10 | SubtitleStep | 字幕生成与嵌入 | Auto |
| 11 | ExportStep | 最终合成输出 | Auto |

### 质量保障机制

每一步都经过 **Quality Gate** 审核：

```
[Step N 输出] → QualityGate 判定
                     │
            ┌────────┴────────┐
            │                 │
         PASS              FAIL
            │                 │
            ▼           [返回 Step N]
       下一 步         自动修复 Prompt
                           │
                           ▼
                      重新执行 Step N
```

## ❓ 常见问题

### Q1: Autonomous Mode 和 Manual Mode 有什么区别？

**Autonomous Mode** 是全自动模式，你只需提供原材料（小说/剧本），AI 会自主完成全部制作流程，你只需要等待结果。

**Manual Mode** 是半自动模式，每一步都需要你审批和确认，你可以随时调整和修改。

### Q2: 制作一个漫剧需要多长时间？

取决于输入内容的长度和复杂度。一般来说：
- 短篇小说（1-3 万字）：约 15-30 分钟
- 中篇小说（5-10 万字）：约 30-60 分钟
- 长篇小说（10 万字以上）：约 1-2 小时

Autonomous Mode 比 Manual Mode 快 3-5 倍。

### Q3: 如果某一步骤质量不通过会怎样？

AI 会自动触发 **Self-Review Loop**，使用修复 Prompt 重新生成该步骤的输出。最多循环 3 次，3 次仍不通过则记录日志并继续执行后续步骤（不阻塞流程）。

### Q4: 制作过程中可以暂停吗？

可以！Autonomous Mode 支持**断点续传**，你可以在任何时候暂停任务，刷新页面后可以继续执行。
