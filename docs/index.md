# 欢迎使用 PanelFlow

PanelFlow 是一个 AI 驱动的专业视频脚本创作平台，帮助你从创意到成品的完整工作流。

## 主要功能

- **视频分析**: 支持多种视频格式，自动识别场景
- **AI脚本生成**: 根据视频内容自动生成专业级脚本
- **分镜头设计**: 自动生成分镜头，支持多种风格
- **角色设计**: AI驱动的角色生成与管理
- **配音合成**: 支持TTS语音合成
- **一键导出**: 支持多种格式导出

## 快速开始

```bash
# 克隆项目
git clone https://github.com/Agions/PanelFlow.git

# 安装依赖
pnpm install

# 启动开发服务器
pnpm tauri dev
```
