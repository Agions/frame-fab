## 主要功能

Panel Flow Refactor 提供以下核心功能：

1. **智能面板重构** - 自动分析并优化面板布局
2. **自动模式** - 无需人工干预的全自动重构流程
3. **工作流程管理** - 清晰的任务执行流程
