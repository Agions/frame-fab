# 角色设计

> 第 4 步：创建角色设定卡 + AI 参考图

## 角色设定卡

每个角色包含：

| 字段   | 说明                    |
| ------ | ----------------------- |
| 姓名   | 角色名字                |
| 性别   | 男/女/其他              |
| 年龄   | 年龄段                  |
| 外貌   | 发型/服装/体态          |
| 性格   | 核心性格特征            |
| 标志   | 独特标识（疤痕/饰品等） |
| 参考图 | AI 生成的角色参考图     |

## AI 参考图生成

基于角色描述自动生成参考图：

```
输入: "叶文洁，中年女性，短发，穿着 60 年代军装，目光坚定"
输出: 角色参考图（ Seedance / Kling / vidu 生成）
```

## 角色一致性

::: tip VLM 比对
后续步骤（分镜/渲染）会使用 VLM（视觉语言模型）比对新生成图像与角色参考图，确保角色一致性 ≥ 0.85。
:::
