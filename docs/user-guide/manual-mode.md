# Manual 模式

> 手动创作模式 — 逐步审批、精细调整

## 适用场景

- 精细定制：每一步都人工把关
- 创意控制：注入自定义 Prompt、替换中间结果
- 学习调试：了解 AI 创作的每一步

## 执行流程

```
导入小说 → [暂停] 人工审批 → 分析 → [暂停] 人工审批 → 剧本 → ... → 成片
```

每一步完成后暂停，等待用户：

- **通过** → 进入下一步
- **重试** → 重新执行当前步骤（可修改 Prompt）
- **编辑** → 直接修改中间结果
- **跳过** → 跳过当前步骤（使用默认值）

## 与 Autonomous 对比

| 操作        | Autonomous     | Manual             |
| ----------- | -------------- | ------------------ |
| 步骤间切换  | 自动           | 手动点击「下一步」 |
| 失败处理    | 自动重试 ≤3 次 | 提示用户决定       |
| Prompt 调整 | 使用默认       | 每步可自定义       |
| 中间结果    | 不可编辑       | 可直接编辑         |
| Checkpoint  | 30s 自动保存   | 手动保存           |

## 自定义 Prompt

每步可注入自定义 Prompt：

```typescript
// 示例：自定义剧本生成 Prompt
const customPrompt = `
基于以下故事结构生成视频剧本：
- 风格：电影感、暗色调
- 节奏：慢节奏、长镜头
- 目标平台：B站（竖屏 9:16）

故事结构：${JSON.stringify(storyAnalysis)}
`;
```

::: tip 推荐工作流

1. 先用 Autonomous 跑通全流程
2. 在关键步骤（SCRIPT / STORYBOARD）切换到 Manual 精细调整
3. 最终导出前用 Autonomous 完成剩余步骤
   :::
