# 导入与分析

> 第 1-2 步：小说导入 + AI 智能分析

## 导入 (IMPORT)

支持格式：

- 直接粘贴文本
- `.txt` / `.md` 文件导入
- 分章节导入（长篇小说推荐）

系统自动完成：

- 章节切分（基于空行/标题）
- 文本清理（去除非法字符）
- 长度评估（token 估算）

::: tip
单次建议 ≤ 5000 字。超过限制请分章节导入。
:::

## AI 分析 (ANALYSIS)

选择 Provider 后点击「开始分析」，输出包含：

| 输出项   | 说明                 |
| -------- | -------------------- |
| 故事结构 | 起承转合四段式分析   |
| 人物清单 | 主/配角识别 + 关系图 |
| 场景识别 | 场景 + 镜头/景别规划 |
| 情感曲线 | 全剧情感起伏分析     |
